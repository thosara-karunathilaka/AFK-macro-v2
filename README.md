# Sea of Thieves AFK Macro by ThosIya

A simple AFK macro made by Thosiya for Sea of Thieves.  
This macro helps prevent inactivity kick while you are away from the keyboard.

## Features
- Automatic movement/actions
- Lightweight and simple
- Changing key binds
- Easy to use
- Designed for Sea of Thieves

## Requirements
- Windows PC
- Sea of Thieves

## Installation

1. Just download it and unzip the zip file and open and run the Advance AFK macro .exe

## Usage

- Open the game
- Run the macro
- Press the assigned hotkey to start
- Press the stop hotkey to disable the macro
